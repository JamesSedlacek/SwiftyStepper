## Installation

SwiftyStepper is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SwiftyStepper'
```

## Requirements
- iOS 15.0
- Swift 5.0
- Xcode 13.0
